Also, if you want to verify that the graphs provided above are of the stated parameters, you may use the following code:

```py
import numpy as np

def check_par(G, r2, r3):
    Gn = np.array(G)
    if np.linalg.norm(np.sum(np.array(Gn), axis=1) - r2) > 0.3:
        return False
    A = Gn@Gn@Gn
    if np.linalg.norm(np.diag(A)-r3*2) >0.3:
        return False
    return True
```
